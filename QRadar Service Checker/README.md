# QRadar Service Checker
### Installation:
```
  mkdir QRadarServiceChecker
  cd QRadarServiceChecker
  wget https://github.com/UnderATK/IBM-QRadar-SIEM/blob/main/QRadar%20Service%20Checker/service_checker.py
```

### Usage:
```
  python3 service_checker.py [-h] [-H HOST_IP] [-o] [-f] [-v] [--version]

  options:
  -h, --help            show this help message and exit
  -H HOST, --host HOST  Host IP.
  -o, --output          Output details to CSV.
  -f, --fix             Show possible fix.
  -v, --verbose         Verbose the details to show on the screen for each step.
  --version             show program's version number and exit
```
